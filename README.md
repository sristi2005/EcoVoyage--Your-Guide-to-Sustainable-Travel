# EcoVoyage 🌍

### Your Guide to Sustainable Travel

EcoVoyage is a sustainable travel planning platform designed to help tourists make environmentally responsible travel choices. The platform provides eco-friendly destination recommendations, sustainable accommodations, green transportation options, and travel tips to reduce the negative environmental impact of tourism.

---

## 📌 Problem Statement

Many tourists struggle to make sustainable travel choices, leading to negative environmental impacts and contributing to climate change.

Travelers often lack:

* Awareness about eco-friendly travel practices
* Access to sustainable accommodations and transportation
* Reliable information about responsible tourism
* Tools to measure and reduce their carbon footprint

---

## 💡 Solution

EcoVoyage helps travelers:

* Discover sustainable destinations
* Find eco-friendly hotels and transportation
* Learn responsible travel practices
* Reduce their environmental impact
* Support local communities and conservation initiatives

---

# 🚀 Features

## 🌱 Sustainable Destination Discovery

* Explore eco-friendly travel destinations
* Filter locations based on sustainability ratings
* Discover green tourism activities

## 🏨 Eco-Friendly Accommodation Finder

* Find certified sustainable hotels and homestays
* View eco-certifications and sustainability practices
* Compare prices and facilities

## 🚆 Green Transportation Suggestions

* Recommend low-carbon transportation options
* Promote public transport, cycling, and EV rentals
* Provide carbon footprint comparisons

## 📚 Sustainable Travel Tips

* Eco-friendly packing guides
* Waste reduction practices
* Responsible tourism awareness

## ⭐ User Reviews & Ratings

* Travelers can share experiences
* Rate sustainability efforts of hotels and tours
* Build a trusted eco-travel community

## 📊 Carbon Footprint Awareness

* Estimate travel emissions
* Suggest greener alternatives
* Encourage carbon offsetting practices

# ⚙️ Installation

## Clone the Repository

```bash
git clone https://github.com/your-username/ecovoyage.git
```

## Navigate to the Project Folder

```bash
cd ecovoyage
```

## Install Dependencies

### Frontend

```bash
cd frontend
npm install
```

### Backend

```bash
cd backend
npm install
```

---

# ▶️ Running the Project

## Start Backend Server

```bash
cd backend
npm start
```

## Start Frontend

```bash
cd frontend
npm start
```

The application will run on:

```bash
http://localhost:3000
```

# 🧠 How EcoVoyage Works

1. Users search for destinations and travel preferences.
2. EcoVoyage analyzes sustainable travel options.
3. The platform recommends:

   * Eco-friendly accommodations
   * Green transportation
   * Responsible tourism activities
4. Users can compare sustainability ratings and reviews.
5. Travelers make informed eco-conscious travel decisions.
